# radcalc

## Intro

Permet de calculer
* DFG
* Poumon
    * modèle de Brock
    * modèle de Herder
    * temps de doublement
    * arbres décisionnels
        * BTS 2015
        * Fleischner 2017
* Onco
    * RECIST
    * [ ] lugano lymphomes
* [ ] Endoc
    * [x] calcul wash-out surrénales
    * [x] volume testicule
    * [ ] mémo tirads, lirads...
* [x] Vasculaire
    * [x] % sténose

## Versions utilisables (Release)

* Sur tous navigateurs (PC, téléphones) [version en ligne](https://aldoniel.github.io/radcalc/)
